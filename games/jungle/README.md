# Peer Jungle – 斗兽棋（Jungle）P2P 联机对战

**Peer Jungle** 是一个基于 **HTML5**、**Tailwind CSS**、**JavaScript** 和 **PeerJS** 构建的纯前端斗兽棋游戏。它利用 WebRTC 实现点对点连接，让你可以和远方的朋友实时对战，无需任何后端服务器。

---

## ✨ 特性

- 🎮 **完整斗兽棋规则** – 实现棋盘、动物移动、吃子、陷阱、兽穴、狮虎跳河、鼠过河等经典机制。
- 🌐 **点对点联机** – 借助 PeerJS 封装 WebRTC，轻松创建或加入游戏，数据直连，低延迟且安全。
- 📱 **响应式设计** – 使用 Tailwind CSS 构建界面，完美适配桌面与移动设备。
- ⚡ **无需安装** – 纯静态页面，打开浏览器即可游玩。
- 🎨 **简洁现代 UI** – 扁平化设计，棋子使用清晰图标（文字/Emoji），预留 img 资源可自行修改棋子图标，棋盘格视觉友好。
- 🔗 **连接简单** – 一方创建房间生成 **Peer ID**，另一方输入 ID 即可加入对局。

---

## 🧰 技术栈

- **HTML5** – 页面结构
- **Tailwind CSS** – 快速、可定制的样式
- **JavaScript** – 核心游戏逻辑与交互
- **PeerJS** – WebRTC 封装，简化点对点数据通道的建立
