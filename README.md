## 这是一个桌游合集项目

---

## 🧰 技术栈

- **HTML5** – 页面结构
- **Tailwind CSS** – 快速、可定制的样式
- **JavaScript (ES6)** – 核心游戏逻辑与交互
- **PeerJS** – WebRTC 封装，简化点对点数据通道的建立

---
## 项目结构
```
jungle/
├── index.html          # 主入口页面
├── games/              # 游戏目录
├── js/                 # 公用 js 资源
└── css/                # 少量自定义样式（主要使用 Tailwind）
```

## 🙏 致谢
 - PeerJS – 让 WebRTC 变得简单易用
 - Tailwind CSS – 高效的原子化 CSS 框架
